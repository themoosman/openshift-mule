# OpenShift Mule CE Images

## Overview

Collection of Dockerfiles used to create Mule Runtime CE images for use in OpenShift.  

## s2i Builder
For an s2i image please see this [repository](https://github.com/tahonen/mule-esb-community-s2i)

## Testing
These images have been tested on OpenShift 3.7 & 3.9.
